# JCSNetS

[![Build Status](https://travis-ci.com/jcs090218/JCSNetS.svg?branch=master)](https://travis-ci.com/jcs090218/JCSNetS)

JCSNetS is a network framework re-wrap Apache Mina framework to a
general MMO base framework.
